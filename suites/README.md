# Validation Suites

This directory contains modular validation suites built on top of the core
cryptographic logic. Each suite targets a specific class of validation scenarios
used in audits, research, and deterministic testing.

## Structure

- deterministic/ — deterministic test vectors and reproducible validation cases
- fuzzing/ — fuzzing‑based reconstruction and edge‑case exploration
- cross-language/ — cross‑implementation validation across multiple languages
- mathematical/ — formal mathematical proofs, invariants, and consistency checks
